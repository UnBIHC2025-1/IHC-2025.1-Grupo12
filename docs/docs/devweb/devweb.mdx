# Desenvolvimento Web

Cada tópico foi alinhado com referências da **NBR 17225:2025**, critérios da **WCAG 2.2** e orientações do **Guia de Acessibilidade do Governo do Reino Unido**, garantindo conformidade técnica e apoio prático à implementação. O objetivo é tornar conteúdos e interfaces mais inclusivos para pessoas com deficiência, promovendo uma web acessível desde o código até a experiência de uso.

- <CheckListItem>1. Texto alternativo para imagens de conteúdo.</CheckListItem>
