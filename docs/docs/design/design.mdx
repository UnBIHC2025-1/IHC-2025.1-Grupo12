## Aparência

### Cores
<ChecklistItem>1. Fazer uso de cores para ajudar o entendimento ao conteúdo da página e facilitar o acesso.</ChecklistItem>
<ChecklistItem>2. Não fazer apenas o uso de cores para transmitir informações, indicar ações, requistiar respostas ou distingir elementos visuais.</ChecklistItem>
    
    ### Contraste de texto 
<ChecklistItem>3. Todos conteúdo de texto, incluindo imagens de textos, têm constraste de pelo menos 7:1 em relação ao fundo ou entre sí.</ChecklistItem>
<ChecklistItem>4. Todos conteúdo de texto está em tamanho grande e têm relação de constraste de pelo menos 4,5:1 em relação ao fundo ou entre sí.</ChecklistItem>

### Constraste de elemento gráfico
<ChecklistItem>5. Todos componenentes de interface têm relação de contraste de pelo menos 3:1 em relação ao fundo ou entre sí.</ChecklistItem>
<ChecklistItem>6. Todas as partes de um objeto gráficos necessários para compreender o conteúdo têm relação de constraste de pelo menos 3:1 em relação ao fundo, ou entre sí.</ChecklistItem>
<ChecklistItem>7. Todos os indicadores de foco viśivel dos elementos têm uma relação de constraste de pelo menos 3:1 com o fundo ou entre o estado sem foco e o com foco.</ChecklistItem>

## Textos

    ### Espaçamento entre as linhas, letras, paragrafos e blocos de texto
    <ChecklistItem>1. Todos os blocos de texto têm um espaçamento entre as linhas (entrelinhas) de pelo menos 1,5 vez o tamanho da fonte.</ChecklistItem>
    <ChecklistItem>2. Todos parágrafos têm espaçamento entre si de pelo menos 2 vezes o tamanho da fonte.</ChecklistItem>
    <ChecklistItem>3. Todos os blocos de texto têm espaçamento entre as linhas (rastreamento) de pelo menos 0.12 vez o tamanho da fonte</ChecklistItem>
    <ChecklistItem>4. Todos os blocos de texto têm espaçamento entre as palavras de pelo menos 0.16 vez o tamanho da fonte</ChecklistItem>
    <ChecklistItem>5. Todos os blocos de texto estão alinhados à esquerda para idiomas lidos da esquerda para direita.</ChecklistItem>
    <ChecklistItem>6. Todos os blocos de texto estão alinhados à direita para idiomas lidos da direita para esquerda.</ChecklistItem>
    <ChecklistItem>7. Existe um mecanismo para configurar o espaçamento onde não haja perda de conteudo ou funcionalidade </ChecklistItem>
    <ChecklistItem>8. A largura do bloco de texto não ultrapassa 80 caracteres.</ChecklistItem>
    <ChecklistItem>9. Existe um mecanismo para configurar a largura dos blocos de texto e alcançar os 80 caracteres por bloco de texto.</ChecklistItem>
